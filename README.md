# OrbitAI

Travel planning and experience engine with a premium aesthetic.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Node.js API Routes

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Run tests:
   ```bash
   npm run test
   ```

## Deployment
This project is ready for Google Cloud Run. A `Dockerfile` is included in the root.
